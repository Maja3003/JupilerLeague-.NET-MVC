## Features
- Team information page with details about Jupiler League.
- User authentication and registration.
- Display of the league table with team statistics (matches played, wins, draws, losses, goal difference, points).
- Collapsible match details for each team.
- Real-time updates of team statistics when matches are added.

## Technologies used
- ASP.NET Core MVC
- Entity Framework Core
- HTML, CSS, JavaScript
- Bootstrap for styling and responsiveness

## What can be added
- Working Team Browser: Add other team pages to view directly from the homepage.
- Season Selection: Choose different seasons other than 22/23 to view their league tables.
