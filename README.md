## Features
- Team information page with details about Jupiler League
- User authentication and registration
- Ability for users to select and manage favorite teams
- Match results and team statistics display
- Real-time updates to the league table standings

## Technologies used
- ASP.NET Core MVC for the backend
- Entity Framework Core for database management
- Razor Views for dynamic front-end rendering
- Bootstrap for responsive design
- Microsoft SQL Server for data storage

## What can be added
- Working Team Browser: Add other team pages to view directly from the homepage.
- Season Selection: Choose different seasons other than 22/23 to view their league tables.
- Player Statistics: Integrate a section to display player stats such as goals scored, assists, minutes played, etc.
- Advanced Match Filtering: Implement filters for users to search for matches by date, team, or scoreline.
- Notifications: Offer real-time notifications for match starts, goals, and final scores.
- User Prediction: Create a feature where users can predict match outcomes.
